# Miru
## 🔎 Overview 
**Miru** ('to watch' in japanese) is a personal movie and series library to track what you’ve watched and what you plan to watch. I'm starting this as a personal university project but who knows where it will end up... 👀

## 🚀 Getting started
For now, you only need an IDE such as Visual Studio Community or Rider to build & run this project.

## 💡 How to contribute ?
If you wish to contribute to this project, please first discuss the change you wish to make via issue, email, or any other method with the owner of this repository before making a change.
